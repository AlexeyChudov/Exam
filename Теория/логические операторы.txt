В си отсутствует логический тип данных(bool), в логических выражениях используются целочисленные типы данных(оптимально - char).

0 -ложь, любое ненулевое число - истина

char flag = 0;

 В стандарте C99 появился тип _Bool определенный типом char.
Любое ненулевое значение, присвоенное переменной типа _Bool приравнивается 1


В заголовочном файле stdbool.h определен тип bool, и две константы: true = 1, false = 0.

Операторы сравнения в си:
> - сравнение на больше
< - сравнение на меньше
>= - больше либо равно
<= - меньше либо равно
== - сравнение на равенство
!= сравнение на неравенство

Логические операторы в си( в порядке приоритета):
1)НЕ !
2) И &&
3)ИЛИ ||


Тернарный оператор:

int a = 5, b = 7;
int result = a > b ? a : b;
printf("%d", result);//->7

Примеры:

if (a>b) printf("a больше b");

if (b != a) printf("b не равно a")

if (b > a && x < y) printf("Условие выполнено!")

if (!(b > a && x < y)) printf("Условие не выполнено!")
